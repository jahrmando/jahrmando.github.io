Cambiar cifrado de contraseñas en CentOS

Posted at — Dec 4, 2014

Podemos cambiar el algoritmo de cifrado que va usar el PAM en el archivo /etc/passwd.

$ vim /etc/sysconfig/authconfig
PASSWDALGORITHM=value
#
# The password algorithm. The value can be bigcrypt, descrypt, md5, sha256, or sha512. For example:
#
PASSWDALGORITHM=sha512

Happy hacking..
